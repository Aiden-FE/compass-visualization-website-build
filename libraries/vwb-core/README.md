# @compass-aiden/vwb-core

> 可视化系统核心包

## Getting Started

`npm install @compass-aiden/vwb-core` 安装依赖

ESModule usage

```typescript
import { demo } from '@compass-aiden/vwb-core';
demo();
```

Commonjs usage

```typescript
const { demo } = require('@compass-aiden/vwb-core');
demo();
```

## Scripts

`npm run dev` 本地开发模式

`npm run build` 构建项目

`npm run lint` 执行代码 Eslint 检查

`npm run format` 执行代码格式化

`npm run test` 执行单元测试,匹配 `src/**/*.spec.ts`

`npm run clean` 清理构建产物
